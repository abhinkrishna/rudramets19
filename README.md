# rudramets19
Project 'Rudra19' is a website we created for our college Tech-Fest 'Rudra' in March 2019. It has a login option with google account. All events conducted on techfest are listed on events page and users can register for those events they wish to, event coordinates can view their registered participants on a separate login.
